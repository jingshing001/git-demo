# Introduce

網頁設計作品功能介紹:


`貼心提醒:按下Alt+任意鍵,會自動focus輸入框`

1.讓使用者輸入代辦事項,瀏覽器會自動保存,並且可以勾選已完成項目,來可以進行編輯和刪除功能

2.讓使用者輸入縣市名稱,並可以取得該縣市,天氣相關的資料並顯示於頁面.
